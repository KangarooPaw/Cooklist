説明：食材を入力したり、料理名を入力することで
レシピを検索してくれるアプリケーションです。

使用言語：python

制作時間：20時間

使い方：・プルダウンメニュー：主菜、副菜、汁物、お菓子からレシピの種類を選択する。
	・テキストボックス（上）：食材名を入力することで、プルダウンメニューと合わせたレシピを表示する。
	・テキストボックス（下）：料理名を入力することで、その料理のレシピを表示する。
	・検索ボタン：テキストボックスに入力した後に押すことで、レシピを検索する。

注意：上下のテキストボックスに入力が入れられていた場合、上のテキストボックスが優先される。

残っているバグ：タイトルのテキストを更新すると裏に前のテキストが残っている。


〜今後追加したい機能〜
・お気に入り機能
	・ファイルの入出力を行いレシピを保存して置けるようにする。
・栄養価の表示
	・摂取エネルギー等を表示する。
・食材のブラックリスト
	・登録してある食材を使わないレシピのみを表示できるようにする。
・画像認識
	・入力を必要とせず写真から食材を読み込み、レシピを検索できるようにする。
・必要食材を表示
	・足りない食材を表示し、付近の店舗で購入した場合の金額を表示できるようにする。
